# Steps to clone this project

1. Clone Repository
    ```sh
    git clone https://lab.skynetinc.tech/WorldDrknss/Task.Management.git
    ```

2. `cd` into the cloned repository
    ```sh
    cd Task.Management
    ``` 

3. Install dependencies
    ```sh
    cd client && npm install
    ```
4. Run Servers
    * Inside the `client` directory
    ```sh
    npm start
    ```
